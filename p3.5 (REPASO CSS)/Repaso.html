Ejercicios de Repaso de CSS

Realiza los ejercicios del 1-5 y súbelos en la carpeta 3.repaso en Github.

1. Diseño de una Página Web Básica

Objetivo: Practicar la aplicación de reglas CSS para dar estilo a elementos HTML básicos.

Instrucciones: Crea una página web con el siguiente contenido:

  • Un encabezado (<header>) con un título centrado.
  • Una lista (<ul>) con tres elementos: "Inicio", "Servicios" y "Contacto".
  • Un párrafo (<p>) con texto de ejemplo.

Aplica los siguientes estilos usando CSS:

  1. Fondo azul para el encabezado con texto blanco.
  2. Cambia el estilo de la lista para que los elementos se muestren horizontalmente (usa flex layout).
  3. Aplica un margen de 20px al párrafo y cambia el color de su texto a gris oscuro.

2. Tablas con Estilo

Objetivo: Practicar el uso de CSS en tablas HTML.

Instrucciones: Crea una tabla con la siguiente estructura:

NOMBRE Edad Ciudad
Ana 23 Madrid
Carlos 30 Barcelona
Elena 28 Sevilla

Aplica los siguientes estilos con CSS:

  1. Alterna colores de fondo para las filas (blanco y gris claro)
  2. Añade un borde negro de 1px a cada celda.
  3. Alinea el texto de las celdas al centro.
